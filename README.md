# TaskFlow - Kanban доска для управления задачами

TaskFlow — это веб-приложение для организации командной работы по методологии Kanban. Проект разработан в учебных целях для демонстрации навыков бэкенд-разработки на Django.

## Функциональность

- Создание проектов (досок)
- Управление задачами (создание, редактирование, удаление)
- Назначение исполнителей и приоритетов
- Комментарии к задачам
- Профили пользователей с дополнительной информацией
- Админ-панель для управления данными

## Технологический стек

- **Python 3.11** — язык разработки
- **Django 4.x** — веб-фреймворк
- **Django REST Framework** — создание REST API
- **PostgreSQL** — база данных
- **Redis** — брокер сообщений для Celery
- **Celery** — асинхронные задачи
- **Docker / Docker Compose** — контейнеризация
- **Nginx** — веб-сервер для статики
